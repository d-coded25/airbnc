AirBNC API

RESTful AirBNC API

About

AirBNC API is RESTful API that allows interaction with the AirBNC database, requesting AirBNC-related resources such as properties and user reviews.

All Properties

An AirBNC endpoint that allows access to property listings, responding with a list of property-related objects as shown in the example directly below.

GET /api/properties


          {
            properties: [
              {
                property_id: number,
                property_name: string,
                location: string,
                price_per_night: number,
                host: string,
                image: URL
              },
              ...
            ]
          }

Optional Queries:

Note: All optional queries are case-sensitive and accepted values are any valid numbers.

  • maxprice: Filters the list of properties set by a maximum price. Example: maxprice=100
  • minprice: Filters the list of properties set by a minimum price. Example: minprice=100

Status Codes:

  • 200 - OK Successful request.
  • 404 - Path not Found Unknown URL.

Single Property

An AirBNC endpoint that allows access to a single property based on a property id, responding with a single property-related object as shown in the example directly below.

GET /api/properties/:id


          {
            property: { 
              property_id: number, 
              property_name: string,
              location: string, 
              price_per_night: number, 
              description: string,
              host: string, 
              host_avatar: URL, 
              image: URL } 
          }

Required Id:

Note: Id parameter must be a valid numeric id corresponding to an existing property id within the database.

Status Codes:

  • 200 - OK Successful request.
  • 400 - Bad Request Invalid Id.
  • 404 - Property Not Found Valid id but the id does not exist.

Reviews For A Specific Property

An AirBNC endpoint that allows access to reviews based on a property id, responding with a single review-related object as shown in the example directly below.

GET /api/properties/:id/reviews


          { 
            reviews: [ 
              { 
                review_id: string, 
                comment: string, 
                rating: number, 
                guest: string, 
                guest_avatar: URL 
              }, 
              ... 
            ] 
          }

Required Id:

Note: Id parameter must be a valid numeric id corresponding to an existing property id within the database.

Status Codes:

  • 200 - OK Successful request.
  • 400 - Bad Request Invalid Id.
  • 404 - Reviews Not Found Valid id but the id does not exist.

Post A Property Review

An AirBNC endpoint that allows a user to post a review about a property determined by the property id, responding with a single review-related object as shown in the example directly below.

POST /api/properties/:id/reviews


          { 
            review: 
              { 
                review_id: number, 
                property_id: number, 
                guest_id: number, 
                rating: number, 
                comment: string, 
                created_at: timestamp 
              }
          }

Required Id:

Note: Id parameter must be a valid numeric id corresponding to an existing property id within the database.

Status Codes:

  • 200 - OK Successful request.
  • 400 - Bad Request Invalid data or required data not provided.

Delete A Property Review

An AirBNC endpoint that allows a user to delete a review about a property determined by the review id.

Required Id:

Note: Id parameter must be a valid numeric id corresponding to an existing review id within the database.

Status Codes:

  • 204 - No Content Successful request.
  • 400 - Bad Request Invalid Id.
  • 404 - Review Not Found Valid id but the id does not exist.
